# Global CO₂ Emissions Analysis

## Overview
This project analyzes historical CO₂ emissions data to compare emission trends across major countries.  
The focus is on visualizing long-term patterns and differences using interactive charts.

---

## Data
- Source: Our World in Data (OWID)
- The dataset contains annual CO₂ emissions data for countries worldwide.
- Variables used in this project:
  - Country
  - Year
  - CO₂ emissions

---

## What This Project Does
- Loads a real-world climate dataset
- Filters selected countries for comparison
- Cleans missing emission values
- Creates an interactive time-series visualization
- Highlights differences in emission trends across countries

---

## Tools & Libraries
- Python
- Pandas
- Plotly Express

---

## Key Insights
- CO₂ emissions have increased significantly in some countries over time
- Emission trends differ substantially across countries
- Interactive visualization makes trend comparison intuitive
